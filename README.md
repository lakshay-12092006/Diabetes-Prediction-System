# 🩺 Diabetes Prediction System

A Machine Learning-based **Diabetes Prediction System** that predicts whether a person is likely to have diabetes based on key medical parameters. The application provides fast and reliable predictions through an easy-to-use Streamlit web interface.

## 🚀 Live Demo

🔗 https://diabetes-prediction-system-3rpw.onrender.com/

---

## 📌 Features

- Predicts diabetes risk instantly
- Simple and user-friendly interface
- Machine Learning classification model
- Fast and accurate predictions
- Responsive web application
- Easy deployment with Render

## 📊 Input Parameters

The model predicts diabetes based on the following inputs:

- Pregnancies
- Glucose
- Blood Pressure
- Skin Thickness
- Insulin
- BMI (Body Mass Index)
- Diabetes Pedigree Function
- Age

## ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/your-username/Diabetes-Prediction-System.git
```

### Navigate to the Project Folder

```bash
cd Diabetes-Prediction-System
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run the Application

```bash
streamlit run app.py
```
